## Hướng dẫn tải hình:

### Cách 1: Tự tìm và tải
Tìm hình ảnh trên Google Images, tải về và đổi tên theo danh sách trên.

### Cách 2: Dùng Unsplash (miễn phí)
Truy cập https://unsplash.com, tìm từng món và tải về.

### Cách 3: Tạo hình placeholder
Tạo 29 file ảnh bất kỳ và đổi tên theo danh sách.

## Yêu cầu:
- Format: JPG hoặc PNG
- Kích thước đề xuất: 800x600px
- Dung lượng: < 500KB mỗi file
- Tên file: viết thường, không dấu, dùng dấu gạch ngang `-`
